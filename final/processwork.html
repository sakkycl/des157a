Visual Design

For the visual design of my project, I created a new design comp using Adobe Illustrator. In my new design comp, I created a new background without the doodles and kept the grid lines. This creates more white space on my website and allows users to focus on the content of my website more easily. I changed the background color for the main content of my website to a light cream color at 100% opacity rather than using a brown color at 75% opacity so the website is less busy and easier on the eyes. I decided to use the typeface, Poppins, because it is a sans-serif font. I thought using a sans-serif font would be easier on the eyes and improve the readability of my website as well. I increased the line height to improve the legibility of the paragraphs. All changes were made to improve the legibility of my website and allow users to comprehend the contents on the page more easily.
