Overlays Design Pattern Research

Febuary 1, 2024

Both articles offered me different perspectives to consider when using overlays and taught me how to use them more wisely. “Best Practices for Modals / Overlays / Dialog Windows” by Naema Baskanderi describes what overlays are, how they are used, and important elements to consider when implementing them. For example, overlay buttons must be actionable and understandable to avoid confusion such as using “close” or “x” to close the overlay. I can see why Baskanderi would recommend doing this because I have encountered overlays where I had trouble closing it because it was hidden. Having clear labels is something I would like to keep in mind if I use overlays in my future projects in order to create websites that are user-friendly.

“Overuse of Overlays: How to Avoid Missing Lightboxes” by Kathryn Whitenton also discuss about overlays and ask important questions to consider when deciding to use overlays or not. For example, one question Whitenton ask is: “Who is the target audience?” Whitenton claims that if users are on a mobile device, overlays tend to be disorientating and block access to certain content. I have also experienced problems with overlays on my phone so I understand where Whitenton is coming from. If I do design a website with overlays I would like to ensure that it works properly on different screen resolutions and in different device orientations.
